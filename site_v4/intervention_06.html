// Le Problème

État initial : Taux de réalisation préventif 68 % (233 OT réalisés sur 342 planifiés). Arrêts imprévus = 23 % du temps de production. Coût maintenance +12 % vs budget. MTBF site 420 h (benchmark industrie connectique ~650 h).

?

// Diagnostic

Analyse GMAO + supervision 12 mois : 109 ordres de travail reportés. Causes Pareto : 45 % conflit production, 30 % pièces indisponibles (délai 5–7 j), 15 % sous-traitant indisponible.

Classification : absente. Tous équipements traités avec la même priorité — le compresseur d'air 8 bar critique et un éclairage de bureau ont le même préventif.

MTBF par famille : presses injection 380 h (faible), bancs test 620 h (moyen), utilités 850 h (bon). Aucun plan d'action ciblé sur les presses.

✓

// Plan d'action 12 mois

01.
DATA — export GMAO (OT, coûts) + supervision (arrêts, codes). Tableau de bord Excel : tendances, top 5 pannes / mois.
02.
CLASSIFICATION ABC (RCM simplifié) — Classe A (14 %, ~20 machines) critiques : préventif mensuel/bimestriel + stock local. Classe B (32 %, ~45) : trimestriel. Classe C (54 %, ~77) : semestriel/annuel.
03.
PLANNING INTÉGRÉ — fenêtres fixes négociées avec production : samedi matin (A), arrêts trimestriels 8 h, vacances (révisions lourdes). Conflits production 45 % → 8 %.
04.
APPRO — négociation fournisseurs locaux, stock sécurité classe A (capteurs, vannes 5/2, joints). Délai critique 5 j → 2 j.
05.
SOUS-TRAITANCE — regroupement par lots (thermographie, étalonnage, pompes spé). Appel d'offres 3 fournisseurs. –18 % coûts.
06.
KPI — tableau de bord mensuel affiché : % préventif, MTBF / MTTR par classe, coût par tonne produite, top 3 pannes.
07.
PROJETS — 5 projets soumis : 2 variateurs obsolètes, suivi vibration 3 groupes critiques, formation auto 3 techniciens. 3/5 validés et réalisés en interne.
→

// Résultat mesuré (année N+1)

Taux de réalisation préventif : 68 % → 94 % (objectif groupe 90 % dépassé). Arrêts imprévus –28 % (production gagne ~340 h/an de disponibilité). MTBF : 420 → 610 h (+45 %). Coûts : –15 % sous-traitance, –8 % pièces. 3 projets amélioration ROI < 18 mois.
